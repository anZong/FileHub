# Product Overview

FileHub is an intelligent file processing platform that handles images, audio, and video files entirely in the browser. The application provides a freemium SaaS model with tiered memberships (free, premium, enterprise) that control feature access and usage limits.

## Core Features

- **Image Processing**: Background removal, ID photo creation, stamp processing
- **Audio Processing**: Format conversion (MP3, WAV, FLAC, AAC, M4A)
- **Video Processing**: Format conversion (MP4, AVI, MKV, MOV, WMV)

## Key Characteristics

- Client-side processing (no file uploads to servers)
- Privacy-focused architecture
- Usage tracking and membership-based limits
- Chinese language interface (zh-CN)
- Authentication via email/password and OAuth (GitHub, Google)

## Business Model

- Free tier: 1 use per feature, 10MB file limit
- Premium tier: Unlimited feature usage, 100MB file limit
- Enterprise tier: All premium features plus API access, 500MB file limit
